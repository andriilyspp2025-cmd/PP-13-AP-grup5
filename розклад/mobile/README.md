# Rozklad Mobile App

React Native mobile application for the Rozklad scheduling system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure API URL:
   - Edit `src/services/api.ts` and update the `API_URL` to point to your backend server

3. Start the development server:
```bash
npm start
```

4. Run on device:
   - For iOS: `npm run ios`
   - For Android: `npm run android`

## Features

- View daily and weekly schedules
- Receive push notifications for schedule changes
- Submit and track change requests
- Beautiful, native mobile UI

## Requirements

- Node.js 18+
- Expo CLI
- iOS Simulator (for iOS development) or Android Emulator (for Android development)
